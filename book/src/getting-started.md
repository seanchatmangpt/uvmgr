# Getting Started

## Installation

Install uvmgr from PyPI:

```bash
pip install uvmgr
```

## Quickstart

1. **Create a new project:**
    ```bash
    uvmgr new my-awesome-project --fastapi
    ```
2. **Navigate into your project:**
    ```bash
    cd my-awesome-project
    ```
3. **Add a dependency:**
    ```bash
    uvmgr deps add "requests"
    ```
4. **Run tests:**
    ```bash
    uvmgr tests run
    ```
5. **Build your package:**
    ```bash
    uvmgr build dist
    ```
