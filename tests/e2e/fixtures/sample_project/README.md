# Sample Project

This is a sample project used for uvmgr end-to-end testing.

## Features

- Basic CLI application
- Unit tests
- Type hints
- Rich terminal output

## Installation

```bash
uvmgr deps install
```

## Usage

```bash
uvmgr run sample-cli --help
```

## Development

```bash
# Run tests
uvmgr tests run

# Run linting
uvmgr lint check

# Build package
uvmgr build wheel
```
